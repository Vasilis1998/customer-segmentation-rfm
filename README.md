This project focuses on segmenting customers of an online retail dataset to identify behavioral patterns and optimize marketing strategies.
📊 Project Overview
Goal: Group customers based on purchasing behavior.
Methodology: RFM Analysis (Recency, Frequency, Monetary) and K-Means Clustering.
Dataset: Online Retail dataset (Transactions).
🛠️ Tech Stack
Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn.
